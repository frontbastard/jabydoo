## Rosetta translations

`django-admin makemessages --all`
and go to `/rosetta`

## Deployment
- `cd /var/www/`
- `git clone git@github.com:frontbastard/jabydoo.git [DOMAIN_NAME]`
- `chmod +x /nginx/deploy_nginx.sh`
- `sudo deploy.sh`
